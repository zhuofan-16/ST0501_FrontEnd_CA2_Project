## Features
* 1.Responsive design
* 2.Live animations using Javascript
* 3.Spinning day/nice picture according to time
* 4.Fly in and Fade in text animation
* 5.Feedback wall with live update of wall everytime you submit a new feedback
* 6.Search feature
* 7.3D slideshow
* 8.Spinable card
* 9.Typing effect for subtitles
